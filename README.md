## 【重要说明】

1. 此软件无窗口，运行后将自动收到任务栏托盘中。
2. 此软件适用于Windows系统设置中锁屏睡眠等相关功能由组织管理员控制，无法自行修改的场景（如下图所示）。
    ![其中某些设置已由组织隐藏或管理](Images/image.png)
3. 启用后默认通过 `每隔30秒` 自动按下 `Ctrl` 键来保持屏幕不锁定。

## 【快捷键】（可在 appsettings.json 文件中修改）

1. 防止屏幕锁定：`Ctrl + Shift + 0`
